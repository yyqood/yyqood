构造函数

1.实例成员
实例成员就是在构造函数内部，通过this添加的成员。实例成员只能通过实例化的对象来访问。
function Star(name,age) {
//实例成员
this.name = name;
this.age = age;
}
2.静态成员
在构造函数本身上添加的成员，只能通过构造函数来访问
//静态成员
Star.sex = ‘女’;

let stars = new Star(‘小红’,18);
console.log(stars); // Star {name: “小红”, age: 18}
console.log(stars.sex); // undefined

console.log(Star.name); //Star
console.log(Star.sex); //女

通过构造函数创建对象（实例化）

1.how?
function Father(name) {
this.name = name;
}
let son = new Father(‘Lisa’); //新建对象
console.log(son); //Father {name: “Lisa”}
2.过程
(1) 创建一个空对象 son {}
(2) 为 son 准备原型链连接
son.proto = Father.prototype
(3) 重新绑定this，使构造函数的this指向新对象
(4) 为新对象属性赋值
(5) 返回this return this，此时的新对象就拥有了构造函数的方法和属性
3.
(1)在构造函数上直接定义方法，不共享
function Star() {
this.sing = function () {
console.log(‘我爱唱歌’);
}
}
let stu1 = new Star();
let stu2 = new Star();
stu1.sing();//我爱唱歌
stu2.sing();//我爱唱歌
console.log(stu1.sing === stu2.sing);//false
(2)通过原型添加方法，共享

（如果属性存储的是基本类型，不存在共享问题，是否相同，看值的内容）

使用方法时，先看obj中是否存在此方法，若没有，则去obj的原型找，以此往下一个原型找，直至没有报错
