正则表达式

一.创建方法

1.通过构造函数
eg:
var reg = new RegExp("a","i");
(i不区分大小写，g设为全局)
2.通过字面量
eg：
var 变量 = /正则表达式/匹配模式
reg = /a/i;

二.检查方式

console.log(reg.test("bcc"));
//返回true或者false

三.使用细节

  1. |、[]里的内容也表示或的关系

      [ab] == a|b   //true

    [a-z] 任意小写字母 [A-Z] 任意大写字母 [A-z] 任意字母
    [0-9] 任意数字
    eg:创建一个正则表达式检查一个字符串中是否有abc、adcaec
    reg =/a[bde]c/;
    console.log(reg.test("bcc"));

  2. 量词

      -通过量词可以设置一个内容出现的次数
     -量词只对它前边的一个内容起作用
     -{n}正好连续出现n次
     -{n,m}  出现n-m次
     -{n, }  n次以上 
     -/ab+c/  + 至少一个b
     -/ab*c/  * 0个或者多个
     -/ab?c/   0个或1个b

    eg：

     `var reg = /(ab){3}/;`
    //ab正好连续出现3次
 `var reg = /ab{1,3}c/;`
    //b出现1次到3次

  3. 特殊位置

      /^a/  :  ^表示开头
  /a$/  :  $表示结尾
  /^a$/ : 如果在正则表达式中同时使用^ $ 则要求字符串必须完全符合正则表达式

    eg：

     `reg = /^a$/;`
 `reg.test("a");` //true
 `reg.test("aaaaa");` //false 
 `reg.test("aba");`  //false
 `reg = /^a|a$/;`
 `reg.test("aba");`  //true

  4. 特殊字符

      . 表示任意字符
   在正则表达式中使用\表示转义字符
   \.  表示.
   \\ 表示\
   *注意：使用构造函数时，由于他的参数是一个字符串，而\是字符串中的转义字符，如果要使用\，则需要用\\来代替*

    eg：

     `var reg = /\./;`  等价于  `reg = new RegExp("\\.");`
 `console.log(reg.test("2.2.2"));`  //true
 `var reg = /\\/;`  等价于  `reg = new RegExp("\\\\");`

 \w  -任意字母、数字、_   [A-z0-9_]
 \W  -除了字符数字_       [^A-z0-9_
 \d  -任意数字  [0-9]
 \D  -除了数字  [^0-9]
 \s  -空格
 \S  -除了空格
 \b  - 单词边界
 \B  -除了单词边界

    eg：
    reg = /\w/;
    console.log(reg.test("123")); //true
    创建一个正则表达式检查一个字符串是否含有单词

     `reg = /child/;`
 `console.log(reg.test("hello childer"));`  //true
 `reg = /\bchild\b/;`
 `console.log(reg.test("hello childer"));`   //false
 `console.log(reg.test("hello child"));`   //true

四.正则表达式与字符串

1.
split()
-可以将一个字符串拆分为一个数组
-方法中可以【传递一个正则表达式】作以为参数，这样方法将会根据正则表达式去拆分字符串
-这个方法即使不指定全局匹配，也会全都拆分
eg：根据任意字母来讲字符串拆分
var result = str.split(/[a-z]/);
2.
search()
-可以搜索字符串中是否含有指定内容
-如果搜索到指定内容，则会返回第一次出现的索引，如果没有搜索到则返回-1
-它可以接受【一个正则表达式】作为参数，然后会根据正则表达式去检索字符串
-只会查找第一个，即使设置全局匹配也没用
eg：搜索字符串中是否含有 abc adc afc
result = str.search(/a[bef]c/);
3.
match()将提取出来的字符返回放在数组中
-可以根据正则表达式，从一个字符串中将符合条件的内容提取出来
-默认情况下，match只会找到第一个符合要求的内容，找到以后会停止检索
我们可以设置正则表达式为全局匹配模式，这样就会匹配到所有内容
可以为一个正则表达式设置多个匹配模式，且顺序无所谓
eg：
result = str.match(/[a-z]/ig);
//[‘a’, ‘b’, ‘c’, ‘d’, ‘e’, ‘f’]
4.
replace()
-可以将字符串中指定内容替换为新的内容
- 参数
1.被替换的内容，可以接受一个正则表达式作为参数
2.新的内容
- 默认只会替换第一个
eg：删除指定元素（用空格代替）
result = str.replace(/a/g,"");
//‘123456’

DOM(部分)

一.事件简介

就是用户和浏览器之间的交互行为

二.js代码位置

我们可以在事件对应的属性中设置一些js代码
    eg：onclick ondblclick onmousemove (结构与行为耦合)
    `<button class="wsygan" onclick="alert('你干嘛');">213</button>`
可以为按钮对应事件绑定处理函数的形式来响应事件，script标签中
(这样事件被触发时，其对应的函数将会被调用)
eg：
    `var btn = document.getElementById("wsygan");`
    `btn.onmousemove = function(){alert("nihaidian");};`

    附：
        为window绑定一个onload事件：该事件对应的响应函数将会在页面加载完成之后执行，
        这样可以确保我们的代码执行时，所有的dom对象已经加载完毕
        eg：
            window.onload = function(){
                btn.onclick = function(){
                alert("噶那么");
                }
            };

三.(查)获取节点

获取元素节点 （通过document对象调用）

  1. getElementById()

    -通过id属性获取以一个元素节点

  2. getElementsByTagName()

     -通过标签名获取一组元素节点对象
 -这个方法会给我们返回一个*类数组对象*，所有查询到的元素都会封装到对象中

  3. getElementsByName()

     -通过name属性获取一组元素节点对象
 -这个方法会给我们返回一个*类数组对象* ，所有查询到的元素都会封装到对象中

    获取元素子节点 (通过具体的元素节点调用)

  4. getElementsByTagName()

         -方法，返回当前节点的指定标签名后代节点

  5. childNodes()
    -属性，表示当前节点的所有子节点的
    -根据DOM标签间的空白也会当成文本节点
    -注意：在IE8及以下的浏览器中，不会将空白文本当成子节点，

  6. firstChild()
    -属性，表示当前节点的第一个子节点

  7. lastChild()
    -属性，表示当前节点的最后一个子节点

  8. children 属性可以获取当前元素的的所有子元素
    var cns2 = city.children;

    获取父节点和兄弟节点 (通过具体的元素节点调用)

  9. parentNode
    -属性，表示当前节点的父节点

  10. previousSibling（也可能获取到空白的文本）
    -属性，表示当前节点的 前一个 兄弟节点

    previousElementSibling；  不会获取空白文本

  11. nextSibling
    -属性，表示当前节点的 后一个兄弟 节点

    附加

  12. 获取body标签
    var body = document.getElementsByTagName("body")[0];
    或者
    在document中有一个属性body，它保存的是body的引用
    var body = document.body;
    var html = document.documentElement; //保存的是html的根标签
    var all = document.all; //代表的是页面中所有的元素

  13. 根据元素的class属性值查询一组元素节点对象
    getElementsByClassName() 不兼容ie8及其以下的内容

  14. 使用元素选择器
    document.querySelector()

     -需要一个选择器的字符串作为参数，可以根据一个css选择器来查询一个元素节点对象
 (使用该方法，总会返回一个*唯一*的元素，如果满足条件的元素有多个，则只会返回*第一个*)

    document.querySelectorAll()

     -该方法和querySelector()用法类似，不同的是它会将符合条件的元素封装到一个数组中返回
 即使符合条件的元素只有一个，也会*返回数组*

    eg:获取class为box1中的所有div

     `var div = document.querySelector(".box1 div");`

    四.增加节点

  15. document.createElement()

     可以用来创建一个元素节点对象
 它需要一个标签名作为参数，将会根据标签名创建元素节点对象，
 并将创建好的对象作为返回值返回

    eg

    `var li = document.createElement("li");`

    document.createTextNode()

     可以用来创建一个文本节点对象
 需要一个文本内容作为参数，将会根据该内容创建文本节点，并将新的节点返回

    eg

    `var gzText = document.createTextNode("广州");`

  16. appendChild()

     -向一个父节点中添加一个新的子节点
 -用法：父节点.appendChild(子节点)

    eg

     `li.appendChild(gzText);`

  17. insertBefore()

     -可以在指定的子节点前插入新的子节点
 -语法：
 父节点.insertBefore(新节点，旧节点);

    eg

     `city.insertBefore(li,bj);`

  18. replaceChild()

     -可以使用指定的子节点替换已有的新节点
 -语法：
     父节点.replaceChild(新节点，旧节点);

    eg

     `city.replaceChild(li,bj);`

  19. removeChild()

     -可以删除指定的子节点
 -语法：
     父节点.removeChild(子节点)
     子节点.parentNode.removeChild(子节点)

    eg

     `city.removeChild(bj);`

附加 1. bj.parentNode.removeChild(bj); //更常用，不用查找父元素
2. 使用innerHTML也可以完成DOM的增删改
city.innerHTML += "<li>广州</li>";

 }
  5. this

  6. confirm()
    用于弹出一个带有确认和取消按钮的提示框

    如果用户点击确认，则返回true，如果点击取消则返回false
